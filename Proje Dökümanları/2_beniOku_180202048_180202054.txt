----------------------------------------------Projeyi Çalıştırma-------------------------------------------------------------
Proje klasorü açıldığında 
Prolab 6 -> BağlantıBilgileri.txt
dosyasına giderek sırasıyla "datasource","port","username","password" bilgilerinizi var olan yerlere yazınız.
MySQL Workbench'e girin. MySQL Connections'tan giriş yapın.girdikten sonra .sql dosyanısını çalıştırmak
için sırasıyla;
file -> Open SQL Script 
Diyerek .sql dosyasını bulunduğu konumdan seçerek ekleyiniz.
Eklendikten sonra Projenin bulunduğu lasöre giderek
Prolab 6 -> Prolab 6.sln konumundan projeyi çalıştırın.
Visual Studio Code açıldıktan sonra Başlat tuşuna basarak projeyi çalıştırın.
